## How to maintain the page

Here are some things to have in mind when entering data in the google sheets
 - When pasting the abstract or title into the sheet, make sure to paste without `style', just plain text.
 - There is LaTeX support, use dollar signs as in $\alpha = 2$ to get equations to appear. That also applies for titles.
 - Make sure that that the date is correctly set.
 - If the abstract contains the dollar sign (as in the monetary amount) you need to escape it via () to avoid it being treated as a LaTeX command.
 - If the abstract needs multiple paragraphs, add \<br/>\<br/> to the end of a paragraph. A single break will not have the desired effect. A paragraph break in the google sheet cell is not enough to make the break appear in the page.
 - Links should be tagged explicitly, for instance, <a href="arxiv>arXiv:1231231</a>
